# FileVault - ASP.NET File Sharing Portal

FileVault is a web-based file sharing portal built using ASP.NET and SQL Server.  
It allows users to upload, manage, and share files through a simple and secure interface.

## 🚀 Live Demo
🌐 Hosted Project: http://www.filevault.somee.com

## 📌 Features
- User Registration & Login
- Secure Authentication
- File Upload & Download
- File Management System
- Database Connectivity using SQL Server
- Session Handling
- Responsive Multi-Page Layout
- Hosted Online using Somee

## 🛠️ Technologies Used
- ASP.NET Web Forms
- C#
- SQL Server
- HTML
- CSS
- JavaScript
- Bootstrap

## 📂 Project Structure
- `Login.aspx` - User login page
- `Register.aspx` - User registration page
- `Dashboard.aspx` - Main dashboard
- `Upload.aspx` - File upload functionality
- `Web.config` - Application configuration
- `App_Data` - Database related files

## ⚙️ Setup Instructions
1. Clone the repository
2. Open the project in Visual Studio
3. Configure SQL Server connection string in `Web.config`
4. Restore database
5. Run the project using IIS Express
